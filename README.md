# 🌦️ WeatherAPI

WeatherAPI provides real-time and forecast weather data for the **LifeOfBees** application, including daily temperature, wind speed, and precipitation levels.

---

## 🐝 Purpose

The API supplies essential weather information that directly impacts bee activity and hive management within the game.

Weather conditions influence:

* Bee productivity
* Honey production
* Seasonal preparation
* Overall hive health

---

## 🔌 Integration

Weather data is consumed by the LifeOfBees backend via Spring’s RestTemplate and exposed to the application to simulate realistic environmental conditions.

---

Designed to enhance realism and bring dynamic environmental behavior into the LifeOfBees ecosystem.
